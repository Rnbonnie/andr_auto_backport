# Autostart Video Backport (Android 7.1.2)

Это приложение автоматически запускает видео при загрузке системы, адаптированное для Android 7.1.2 (Nougat, API 25).

## Особенности
- Поддержка Android 7.1.2 (API 25).
- Автозапуск при загрузке системы (`BOOT_COMPLETED`).
- Удержание фокуса через 3 секунды после загрузки.
- Циклическое воспроизведение видео.
- Поиск видео в папках:
    - `/sdcard/autostart/` (любой видеофайл)
    - `/sdcard/Movies/auto.mp4`
    - `/sdcard/Download/auto.mp4`

## Требования
- Разрешение на чтение внешнего хранилища.
- Разрешение "Отображение поверх других окон" (рекомендуется для стабильного автозапуска).

## Сборка
Для сборки можно использовать `gradle` или `nix-shell`:
```bash
nix-shell --run "gradle assembleDebug"
```
APK будет находиться в `app/build/outputs/apk/debug/`.
